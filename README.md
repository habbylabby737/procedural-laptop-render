# Procedural 3D Laptop Render

This is a self-contained WebGL 2 render inspired by `/Users/matthewrini/Desktop/Image 3.jpeg`.

It recreates the scene procedurally: a MacBook-style laptop with black bezel, top notch, soft blue-green lock-screen wallpaper, centered Codex lock prompt, foreground keyboard, warm desk, glass, book/pen, and blurred window/greenery background.

Open `index.html` directly, or serve the folder locally:

```sh
python3 -m http.server 8080 --bind 127.0.0.1
```
